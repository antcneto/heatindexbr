# Package index

  : Extract Heat Index for municipalities of the Brazilian Semiarid
- [`hi_plot()`](https://antcneto.github.io/heatindexbr/reference/hi_plot.md)
  : Plot a Heat Index raster
- [`hi_search()`](https://antcneto.github.io/heatindexbr/reference/hi_search.md)
  : Search for municipalities in the Brazilian Semiarid
- [`hi_shape()`](https://antcneto.github.io/heatindexbr/reference/hi_shape.md)
  : Extract Heat Index for a custom spatial area
- [`semiarido_municipios`](https://antcneto.github.io/heatindexbr/reference/semiarido_municipios.md)
